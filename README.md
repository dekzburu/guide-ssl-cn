# guide-ssl-cn

## 项目简介

本项目用于归档和发布多个独立的 HTML 页面，内容主要围绕 SSL/TLS 相关技术信息的整理与展示。所有页面均为静态 HTML 文件，不依赖后端服务，可直接通过浏览器访问。

本仓库不针对任何特定域名、网站或服务，仅作为通用技术资料的存档与发布渠道。

## 目录结构

```
├── README.md          # 本文件
├── index.html         # 项目主页入口
├── pages/             # 存放各独立 HTML 页面
│   ├── page1.html
│   ├── page2.html
│   └── ...
└── assets/            # 静态资源（CSS、图片等）
    ├── css/
    └── images/
```

## 页面归档说明

- 所有页面均为独立 HTML 文件，可单独访问。
- 页面内容以 SSL/TLS 相关技术信息为主，包括但不限于配置参考、常见问题、技术对比等。
- 页面不包含任何商业推广、诱导点击或特定服务推荐内容。
- 页面可能随时间更新或新增，具体以仓库最新提交为准。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接在浏览器中打开 `index.html` 或 `pages/` 目录下的任意 HTML 文件。
3. 也可通过 GitHub Pages 或其他静态托管服务部署后访问。
